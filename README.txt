# `src`目录说明

本目录存放课程设计的所有源代码。

## 主要子目录和文件说明

- `lib/`
  - 存放调用的函数包/模组
- `upsampleSignal.py`
  - 实现真实语音信号增采样，调用说明请看`upsampleSignal使用说明.html`或者`upsampleSignal使用说明.md`
- `unit-test-upsample.ipynb`
  - 实现真实语音信号增采样单元测试函数
- `workbench.ipynb`
  - "数字信号处理"课程设计——工作台
